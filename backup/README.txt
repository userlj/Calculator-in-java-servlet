This project is a base 2 calculator.


I use jsp and Java Servlet.


There are 12 buttons:

	0,1--binary digit
	  C--clear accumulator
	 MC--clear internal memory
	 MR--display the contents of memory
	 MS--store the content of accumulator in memory
	  +--addition operation
	  ---substraction operation
	AND--bitwise AND operation
	 OR--bitwise OR operation
	XOR--bitwise XOR operation
	  =--display result

This calculator works for up to 8 bits entris, if the input operands
or the results exceed 8 bits, an error message will show up.
